# Definitions and Notation

## Reasoning Trajectory Space

**Definition 1 (Reasoning State).** A *reasoning state* is an element s ∈ Σ* (finite string over alphabet Σ), representing a snapshot of a logical argument at a particular step. In the continuous setting, we embed reasoning states in a Banach space E (e.g., ℝ^d for d-dimensional token embeddings).

**Definition 2 (Reasoning Trajectory).** A *reasoning trajectory* of length n is a finite sequence τ = (s₀, s₁, ..., sₙ) ∈ (Σ*)^{n+1}, where s₀ is an initial query/premise and each sᵢ represents an intermediate reasoning step.

**Definition 3 (Trajectory Space).** The *trajectory space* is (M, d) where:
- M = ⋃_{n≥0} E^{n+1} is the set of all finite-length trajectories in the Banach space E
- d: M × M → [0, ∞) is a metric, defined below

**Definition 4 (Trajectory Metric).** For trajectories τ = (s₀, ..., sₘ) and τ' = (s₀', ..., sₙ') in M, define:
  d(τ, τ') = |m - n|·δ + min(m,n)⁻¹ · Σᵢ₌₀^{min(m,n)} ‖sᵢ - sᵢ'‖_E

where δ > 0 is a length-mismatch penalty. When restricted to equal-length trajectories M_n = E^{n+1}, this reduces to the normalized ℓ¹ metric.

**Remark.** For the main theoretical results, we work primarily with fixed-length trajectories M_n, which is a complete metric space when E is a Banach space.

## CoT Operators

**Definition 5 (CoT Operator).** A *chain-of-thought operator* is a mapping T_p: M_n → M_n parameterized by a prompt p ∈ P (the prompt space), defined by:
  T_p(s₀, s₁, ..., sₙ) = (s₀, f_p(s₀, s₁), f_p(s₁, s₂), ..., f_p(sₙ₋₁, sₙ))

where f_p: E × E → E is the *transition function* induced by prompt p. The first component s₀ (the query) is preserved.

**Definition 6 (Lipschitz Constant of a CoT Operator).** The Lipschitz constant of T_p is:
  Lip(T_p) = sup_{τ ≠ τ'} d(T_p(τ), T_p(τ')) / d(τ, τ')

T_p is *contractive* if Lip(T_p) < 1, *non-expansive* if Lip(T_p) ≤ 1, and *expansive* if Lip(T_p) > 1.

## Semigroup Structure

**Definition 7 (Reasoning Trajectory Semigroup).** The *reasoning trajectory semigroup* is:
  S = S(P) = {T_{pₖ} ∘ ⋯ ∘ T_{p₁} : pᵢ ∈ P, k ≥ 1}

equipped with composition as the binary operation. When we adjoin the identity operator Id, we obtain the *reasoning trajectory monoid* S¹.

**Definition 8 (Homogeneous Iteration Semigroup).** For a single prompt p, the *homogeneous iteration semigroup* is:
  S_p = {T_p^n : n ≥ 1} = {T_p, T_p², T_p³, ...}

This is the cyclic semigroup generated by T_p.

**Definition 9 (Green's Relations).** For elements a, b ∈ S:
- a ℒ b iff S¹a = S¹b (same left ideal)
- a ℛ b iff aS¹ = bS¹ (same right ideal)
- a 𝒥 b iff S¹aS¹ = S¹bS¹ (same two-sided ideal)
- a ℋ b iff a ℒ b and a ℛ b
- a 𝒟 b iff ∃c: a ℒ c and c ℛ b

## Convergence

**Definition 10 (Operator Convergence).** A sequence (Sₙ)_{n≥1} in S *converges* if for every τ ∈ M_n, the sequence (Sₙ(τ))_{n≥1} converges in (M_n, d). The *limit operator* is T_∞(τ) = lim_{n→∞} Sₙ(τ).

**Definition 11 (Fixed-Point Reasoning Pattern).** A trajectory τ* ∈ M_n is a *fixed-point reasoning pattern* for operator T ∈ S if T(τ*) = τ*. The set of fixed points is Fix(T) = {τ ∈ M_n : T(τ) = τ}.

**Definition 12 (Idempotent Element).** An element e ∈ S is *idempotent* if e² = e. The set of idempotents is E(S).

**Definition 13 (Semigroup at Infinity).** The *semigroup at infinity* is:
  S_∞ = ⋂_{k≥1} cl{Sₙ : n ≥ k}

where the closure is in the strong operator topology.

## Spectral Properties

**Definition 14 (Linearization).** For a differentiable CoT operator T_p at fixed point τ*, the *linearization* is DT_p(τ*): E^{n+1} → E^{n+1}, the Fréchet derivative.

**Definition 15 (Spectral Radius).** The *spectral radius* of the linearization is:
  ρ(DT_p(τ*)) = lim_{k→∞} ‖DT_p(τ*)^k‖^{1/k}

Convergence of T_p^n near τ* is characterized by ρ(DT_p(τ*)) < 1.
